Create app/core/schema.py with these exact 7 SQLAlchemy 2.x declarative models using Mapped type annotations:

Model: User (__tablename__ = "users")
Columns: id (UUID PK, default uuid4), username (String 64, unique), email (String 255, unique), hashed_password (String 255), is_active (Boolean default True), created_at (DateTime tz-aware), updated_at (DateTime tz-aware, onupdate)
Relationship: user_roles -> UserRole

Model: Role (__tablename__ = "roles")
Columns: id (UUID PK), name (String 64, unique — values: "chef"/"gm"/"accountant"), description (Text nullable)
Relationship: user_roles -> UserRole

Model: UserRole (__tablename__ = "user_roles")
Columns: id (UUID PK), user_id (UUID FK users.id CASCADE), role_id (UUID FK roles.id CASCADE), location_id (UUID NULLABLE — no FK, future use), assigned_at (DateTime)
Constraint: UniqueConstraint("user_id", "role_id", "location_id", name="uq_user_role_location")
NO boolean role fields — the join table IS the role assignment.

Model: Vendor (__tablename__ = "vendors")
Columns: id (UUID PK), name (String 255), normalized_name (String 255, index=True — lowercased stripped for fuzzy match), contact_email (String 255 nullable), created_at (DateTime)

Model: GlAccount (__tablename__ = "gl_accounts")
Columns: id (UUID PK), code (String 32, unique), name (String 255), account_type (String 64 — "expense"/"asset"/"liability"), is_active (Boolean default True), created_at (DateTime)

Model: AuditLog (__tablename__ = "audit_log")
Columns: id (UUID PK), user_id (UUID nullable — system actions have no user), action (String 255), entity_type (String 128 nullable), entity_id (String 255 nullable), payload (Text nullable — JSON), created_at (DateTime)

Model: PluginRegistry (__tablename__ = "plugin_registry")
Columns: id (UUID PK), name (String 128, unique — plugin name is the unique key for upsert), version (String 32), description (Text default ""), installed_at (DateTime), updated_at (DateTime, onupdate)

All models import from: sqlalchemy (String, Text, DateTime, ForeignKey, UniqueConstraint, Boolean, Uuid), sqlalchemy.orm (DeclarativeBase, Mapped, mapped_column, relationship). Use uuid.UUID type hint + Uuid column type. All DateTime columns use timezone=True.
